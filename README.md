# 🛒 Ecommerce AI Price Negotiation System

> Smarter Shopping with AI-Powered Price Negotiation 🤖

A smart Java-based eCommerce application that allows users to browse products, negotiate prices using AI, and complete purchases with modern features like QR payments, wishlist, and price alerts.

---

## 🚀 Features

### 🤖 AI Price Negotiation
- Chat-based negotiation with AI
- Smart discount logic (max 25%)
- Real-time deal acceptance system
- Final negotiated price applied only to cart

### 🛍️ Product Catalog
- Browse all products with images
- Stock and pricing details
- Clean UI design

### 🛒 Shopping Cart
- Add/remove/update items
- Real-time total calculation
- Negotiated price support

### 💳 Checkout System
- Cash on Delivery
- UPI QR Code Payment (auto-generated with amount)

### 📦 Order Management
- View all previous orders
- Expandable order cards
- Delivery tracking timeline UI

### ❤️ Wishlist
- Save products
- Quick add to cart

### 📉 Price Drop Alerts
- Set alert for price drops
- Notification system

### 🎯 Mood-Based Shopping
- Filter products by mood (Gaming, Work, Chill, etc.)

---

## 🛠️ Tech Stack

- **Frontend:** Java Swing (GUI)
- **Backend:** Core Java (DAO + Service Layer)
- **Database:** MySQL
- **API:** OpenRouter (AI Negotiation)
- **Other:** JDBC, UPI QR Integration

---

## 🧠 Key Highlights

- AI-powered real-time negotiation 🤖  
- Secure authentication with hashed passwords 🔐  
- Clean layered architecture (DAO, Service, UI)  
- Scalable & modular design
